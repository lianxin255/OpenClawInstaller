# 每日报告技能

## 描述
生成每日工作报告和待办事项总结，帮助你回顾一天的工作成果。

## 触发条件
- 用户说 "生成日报" 或 "每日报告" 或 "今日总结"
- 每天晚上 20:00 自动触发（可在配置中调整）

## 执行步骤
1. 收集今日完成的任务和对话摘要
2. 列出未完成的待办事项
3. 总结今日的重要信息和决策
4. 生成明日工作计划建议
5. 统计今日 API 使用量（可选）

## 输出格式

```
📅 {日期} 每日报告

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ 今日完成
{完成的任务列表，带时间戳}

📋 待办事项
{未完成的任务，按优先级排序}

💡 重要信息
{今日的关键信息和决策}

📊 统计数据
• 对话次数: {count}
• 处理任务: {count}
• API 消耗: {tokens} tokens

🎯 明日建议
{基于今日情况的明日计划建议}

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 配置选项

```yaml
skills:
  daily-report:
    enabled: true
    schedule: "0 20 * * *"  # 每天 20:00 触发
    include_stats: true      # 包含统计数据
    include_suggestions: true # 包含明日建议
    channels:                # 推送渠道
      - telegram
      - discord
```
